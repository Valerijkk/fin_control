# Критерии приёмки: 06 — Sentry

## Обязательно

- [ ] Создан проект в sentry.io (или использован существующий), получен DSN для Flutter.
- [ ] DSN подставлен в `lib/config/student_env.dart` (константа `sentryDsn`), Sentry инициализирован в `main.dart`.
- [ ] В настройках приложения при наличии DSN отображается кнопка «Тест Sentry» (или тестовое исключение вызвано иным способом).
- [ ] После нажатия «Тест Sentry» (или краша) событие/краш появилось в дашборде Sentry (с учётом задержки отправки при перезапуске).
