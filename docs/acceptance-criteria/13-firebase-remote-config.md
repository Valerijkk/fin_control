# Критерии приёмки: 13 — Firebase Remote Config

**Практика:** [13 — Firebase Remote Config](../practices/13-firebase-remote-config.md)

Приложение FinControl поставляется из коробки. Для Firebase используются конфиги проекта (`google-services.json` / `GoogleService-Info.plist`). См. [00-firebase-setup](00-firebase-setup.md), [STUDENT_ENV.md](../STUDENT_ENV.md).

## Обязательно

- [ ] Выполнена практика 00-firebase-setup.
- [ ] В Firebase Console создан хотя бы один параметр Remote Config.
- [ ] В приложении подключён Remote Config, значение параметра запрашивается (fetch) и используется в UI или логике.
- [ ] Изменение значения параметра в консоли и обновление (fetch) в приложении приводят к ожидаемому изменению поведения/отображения.

## Что показать на созвоне

Параметр в консоли Remote Config; приложение получает значение и использует в UI или логике. Подробнее: [exam-and-submission.md](../exam-and-submission.md#3-что-показать-и-что-сказать-на-экзамене-по-блокам).
