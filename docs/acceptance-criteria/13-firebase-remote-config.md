# Критерии приёмки: 13 — Firebase Remote Config

## Обязательно

- [ ] Выполнена практика 00-firebase-setup.
- [ ] В Firebase Console создан хотя бы один параметр Remote Config.
- [ ] В приложении подключён Remote Config, значение параметра запрашивается (fetch) и используется в UI или логике.
- [ ] Изменение значения параметра в консоли и обновление (fetch) в приложении приводят к ожидаемому изменению поведения/отображения.
