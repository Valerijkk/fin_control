# Критерии приёмки: 11 — Firebase FCM

**Практика:** [11 — Firebase FCM](../practices/11-firebase-fcm.md)

Приложение FinControl поставляется из коробки. Для Firebase используются конфиги проекта (`google-services.json` / `GoogleService-Info.plist`). См. [00-firebase-setup](00-firebase-setup.md), [STUDENT_ENV.md](../STUDENT_ENV.md).

## Обязательно

- [ ] Выполнена практика 00-firebase-setup.
- [ ] Подключён Firebase Messaging, запрошены разрешения на уведомления (где требуется).
- [ ] FCM-токен получен и доступен (выведен в лог или на экран настроек).
- [ ] Из Firebase Console (Cloud Messaging) отправлено тестовое сообщение на этот токен.
- [ ] Уведомление получено на устройстве и отображается (в foreground/background по реализации).

## Что показать на созвоне

FCM-токен в логах или настройках; тестовое push из консоли приходит в приложение. Подробнее: [exam-and-submission.md](../exam-and-submission.md#3-что-показать-и-что-сказать-на-экзамене-по-блокам).
