# Критерии приёмки: 15 — Firebase In-App Messaging

**Практика:** [15 — Firebase In-App Messaging](../practices/15-firebase-in-app-messaging.md)

Приложение FinControl поставляется из коробки. Для Firebase используются конфиги проекта (`google-services.json` / `GoogleService-Info.plist`). См. [00-firebase-setup](00-firebase-setup.md), [STUDENT_ENV.md](../STUDENT_ENV.md).

## Обязательно

- [ ] Выполнена практика 00-firebase-setup.
- [ ] Firebase In-App Messaging подключён в приложении.
- [ ] В Firebase Console создана тестовая кампания In-App Messaging (триггер — событие или экран).
- [ ] При выполнении условия показа (например открытие экрана Обменник) сообщение отображается в приложении.

## Что показать на созвоне

Тестовая кампания в консоли; сообщение показывается в FinControl при выполнении триггера. Подробнее: [exam-and-submission.md](../exam-and-submission.md#3-что-показать-и-что-сказать-на-экзамене-по-блокам).
