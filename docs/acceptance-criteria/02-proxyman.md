# Критерии приёмки: 02 — Proxyman

**Практика:** [02 — Proxyman](../practices/02-proxyman.md)

Приложение FinControl поставляется из коробки. Для практик Sentry, AppMetrica и Firebase ключи подставляются в указанный файл: `lib/config/student_env.dart` (Sentry, AppMetrica) или конфиги Firebase в проекте. См. [STUDENT_ENV.md](../STUDENT_ENV.md).

## Обязательно

- [ ] Proxyman установлен и запущен, прокси на устройстве/симуляторе настроен.
- [ ] Сертификат Proxyman установлен для расшифровки HTTPS.
- [ ] FinControl открыт, выполнен переход на экран с загрузкой курсов.
- [ ] В Proxyman виден запрос к API курсов (exchangerate.host или open.er-api.com).
- [ ] Ответ успешный (200), тело — JSON с курсами.

## Желательно

- [ ] Ученик продемонстрировал **Breakpoint**: остановил ответ API, изменил данные, показал результат в приложении.
- [ ] Ученик продемонстрировал **Map Local**: подставил локальный JSON-файл вместо реального ответа.
- [ ] Ученик продемонстрировал **Network Conditions**: имитация медленной сети и проверка поведения (таймаут, кэш).
- [ ] Понимание: перехват HTTPS с сертификатом Proxyman — только для учебной среды; не использовать на устройствах с чувствительными данными (см. [practices/02-proxyman.md](../practices/02-proxyman.md), блок «Безопасность»).

## Что показать на созвоне

Proxyman перехватывает трафик FinControl; виден запрос к API курсов и ответ с JSON. Подробнее: [exam-and-submission.md](../exam-and-submission.md#3-что-показать-и-что-сказать-на-экзамене-по-блокам).
