# Критерии приёмки: 12 — Firebase Analytics

**Практика:** [12 — Firebase Analytics](../practices/12-firebase-analytics.md)

Приложение FinControl поставляется из коробки. Для Firebase используются конфиги проекта (`google-services.json` / `GoogleService-Info.plist`). См. [00-firebase-setup](00-firebase-setup.md), [STUDENT_ENV.md](../STUDENT_ENV.md).

## Обязательно

- [ ] Выполнена практика 00-firebase-setup.
- [ ] Firebase Analytics подключён в приложении.
- [ ] Реализовано логирование экранов (screen_view) и при необходимости ключевых событий.
- [ ] Приложение запущено в debug, выполнен сценарий (переходы по экранам).
- [ ] В Firebase Console → Analytics → DebugView (или отчёты) отображаются соответствующие события.

## Что показать на созвоне

Сценарий в приложении (переходы по экранам); события в Firebase Analytics. Подробнее: [exam-and-submission.md](../exam-and-submission.md#3-что-показать-и-что-сказать-на-экзамене-по-блокам).
