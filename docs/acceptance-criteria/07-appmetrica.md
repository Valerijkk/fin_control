# Критерии приёмки: 07 — AppMetrica

**Практика:** [07 — AppMetrica](../practices/07-appmetrica.md)

Приложение FinControl поставляется из коробки. Для AppMetrica API Key подставляется в указанный файл: `lib/config/student_env.dart` (константа `appMetricaApiKey`). Реальный ключ не коммитить. См. [STUDENT_ENV.md](../STUDENT_ENV.md).

## Обязательно

- [ ] Приложение FinControl заведено в кабинете AppMetrica, получен API Key.
- [ ] SDK AppMetrica подключён в проекте, API Key задан в `lib/config/student_env.dart` (константа `appMetricaApiKey`), инициализация выполняется в `main.dart`.
- [ ] Приложение запущено, выполнены переходы по экранам (список, обменник, портфель и т.д.).
- [ ] В отчётах AppMetrica отображаются сессии и/или устройства (с учётом задержки обработки).

## Желательно (безопасность)

- [ ] Ученик понимает: реальный API Key не коммитится в репозиторий; при push в `student_env.dart` оставлены пустые строки (см. [STUDENT_ENV.md](../STUDENT_ENV.md)).

## Что показать на созвоне

Запуск FinControl и переходы по экранам; в кабинете AppMetrica видны сессия и устройство. Подробнее: [exam-and-submission.md](../exam-and-submission.md#3-что-показать-и-что-сказать-на-экзамене-по-блокам).
