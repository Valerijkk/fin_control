# Техническая документация FinControl

**Одно приложение FinControl** (Flutter) используется для всех мобильных и веб-практик: Android, iOS, при необходимости Web. Документы в этой папке согласованы со структурой и кодом в `lib/`. Здесь — архитектура, модели данных, API и инструкции по среде разработки.

| Документ | Описание |
|----------|----------|
| [architecture.md](architecture.md) | Архитектура приложения: слои, структура проекта, потоки данных, интеграции (Sentry, AppMetrica, Firebase), **производительность** (асинхронные операции, метрики, регрессии). |
| [data-models.md](data-models.md) | Модели данных: таблицы SQLite (расходы, обмен, портфель), категории. |
| [api-spec.md](api-spec.md) | Спецификация API: внешние (курсы валют), внутренние эндпоинты при наличии. |
| [ANDROID_STUDIO_LAUNCH.md](ANDROID_STUDIO_LAUNCH.md) | Запуск проекта в Android Studio: почему открывать корень, пошагово, типичные ошибки Gradle и эмулятора. |
