# Практика: AppMetrica — заведение приложения и проверка

**Одно приложение** FinControl. Всё из коробки: нужно только **вставить API Key в `lib/config/student_env.dart`** и перезапустить приложение. Проверка за **1–2 действия:** запусти приложение, походи по экранам 1–2 минуты → в кабинете AppMetrica (Отчёты / Онлайн) через 5–20 минут появятся сессии и устройства.

## Цель

Зарегистрировать одно приложение FinControl в кабинете AppMetrica (Яндекс), получить API Key, вставить его в единственное место в коде — `lib/config/student_env.dart` — и проверить, что сессии и устройства появляются в отчётах AppMetrica после запуска приложения и навигации по экранам.

## Ожидаемый результат

- В кабинете [appmetrica.io](https://appmetrica.io) создано приложение (Android и/или iOS) с именем FinControl; скопирован **API Key**.
- В файле **`lib/config/student_env.dart`** в константу `appMetricaApiKey` вставлен ваш API Key; приложение перезапущено.
- После запуска приложения и действий (Список → Обменник → Портфель) в кабинете AppMetrica в разделах **Отчёты** / **Онлайн** (с задержкой 5–20 минут) появляются данные об установках, сессиях и устройствах.

**Куда вставлять API Key:** только в [**lib/config/student_env.dart**](../../lib/config/student_env.dart) — константа `appMetricaApiKey`. Сохрани файл и перезапусти приложение — больше ничего не нужно. Подробнее: [STUDENT_ENV.md](../STUDENT_ENV.md).

## Что понадобится

- Аккаунт в [AppMetrica](https://appmetrica.io) (Яндекс)
- Проект FinControl с подключённым AppMetrica SDK для Flutter

## Шаг 1: Регистрация приложения в AppMetrica

1. Перейди на [appmetrica.io](https://appmetrica.io) и войди (через Яндекс или партнёрский вход).
2. Создай приложение: **Добавить приложение** → укажи название (например FinControl), платформу **iOS** и/или **Android**.
3. После создания открой карточку приложения и скопируй **API Key** (числовой ключ). Он понадобится для вставки в `student_env.dart`.

**Безопасность:** не коммитьте API Key и конфиги с ключами в репозиторий. Вставляйте API Key только в локальный `student_env.dart`; при push оставляйте константу пустой. См. [STUDENT_ENV.md](../STUDENT_ENV.md) (раздел «Безопасность ключей»).

## Шаг 2: Куда вставить API Key в приложении

1. В проекте FinControl API Key задаётся **в одном файле**. Открой [**lib/config/student_env.dart**](../../lib/config/student_env.dart).
2. Найди константу **`appMetricaApiKey`** (в начале файла капсом подписано: «ПРАКТИКА 07 — APPMETRICA»). Вставь скопированный API Key **в пустую строку между кавычками** (сейчас там `''`). После подстановки будет, например: `const String appMetricaApiKey = 'твой_ключ_из_кабинета';` — не коммить в репозиторий заполненное значение, при push оставляй `''`.
3. Сохрани файл. **Перезапусти приложение** (`flutter run` заново) — инициализация AppMetrica выполняется в `lib/main.dart` при старте при непустом ключе (пакет `appmetrica_plugin` уже подключён в FinControl).
4. Не дублируй API Key в других файлах; добавлять код инициализации вручную не нужно.

## Шаг 3: Проверка за 1–2 действия

1. Запусти приложение после сохранения `student_env.dart` и походи по экранам: Список, Обменник, Портфель (1–2 минуты).
2. В браузере открой кабинет AppMetrica → твоё приложение FinControl.
3. Перейди в раздел **Отчёты** / **Стандартные отчёты** или **Онлайн**.
4. Учитывай задержку обработки данных: **5–20 минут** (при первом запуске может быть дольше). Обнови страницу или зайди позже.
5. В отчётах должны появиться **Установки**, **Сессии**, **Устройства** (модель, ОС, версия приложения).

## Проверка

- [ ] Приложение FinControl заведено в AppMetrica, API Key скопирован из настроек приложения.
- [ ] В **`lib/config/student_env.dart`** в `appMetricaApiKey` вставлен ваш ключ (не пустая строка); файл сохранён, приложение перезапущено.
- [ ] Приложение было запущено и использовалось (навигация по экранам 1–2 минуты).
- [ ] В отчётах AppMetrica видны установки/сессии/устройства (после ожидания 5–20 минут).

## Что показать на экзамене / созвоне

1. Запустить приложение FinControl с подключённым AppMetrica, походить по экранам (Список, Обменник, Портфель).
2. Открыть кабинет AppMetrica → ваше приложение → **Отчёты** / **Онлайн**.
3. Показать, что появились сессии и/или устройства (с учётом задержки обновления данных).
4. Кратко сказать: «Добавил API key AppMetrica в приложение, запустил его и выполнил действия — в отчётах вижу сессии и устройство.»
