# Практика: AppMetrica — заведение приложения и проверка

## Цель

Зарегистрировать приложение FinControl в кабинете AppMetrica (Яндекс), получить API Key, подключить SDK в приложении и убедиться, что сессии и устройства отображаются в отчётах.

## Что понадобится

- Аккаунт в [AppMetrica](https://appmetrica.io) (Яндекс)
- Проект FinControl с подключённым AppMetrica SDK для Flutter

## Шаг 1: Регистрация приложения в AppMetrica

1. Перейдите на [appmetrica.io](https://appmetrica.io) и войдите (через Яндекс или партнёрский вход).
2. Создайте приложение: **Добавить приложение** → укажите название (например FinControl), платформу **iOS** и/или **Android**.
3. После создания откройте карточку приложения и скопируйте **API Key** (числовой ключ). Он понадобится для инициализации SDK.

## Шаг 2: Подключение AppMetrica в Flutter

1. В `pubspec.yaml` добавьте пакет AppMetrica для Flutter (актуальное имя уточните на pub.dev, например `appmetrica_sdk` или `yandex_appmetrica`):
   ```yaml
   dependencies:
     appmetrica_sdk: ^3.0.0  # проверьте актуальную версию
   ```

2. В проекте FinControl API Key задаётся в `lib/config/telemetry.dart` (константа `appMetricaApiKey`). Заполните её — инициализация выполняется в `lib/main.dart` при непустом ключе (пакет `appmetrica_plugin`).

3. Перезапустите приложение и выполните несколько действий (открытие экранов, обмен, портфель), чтобы отправить сессию.

## Шаг 3: Проверка в кабинете AppMetrica

1. В кабинете AppMetrica откройте ваше приложение.
2. Перейдите в раздел **Отчёты** / **Стандартные отчёты** (или **Онлайн**).
3. Подождите несколько минут (данные могут приходить с задержкой).
4. Проверьте:
   - **Установки** — должна появиться установка с вашего устройства/эмулятора.
   - **Сессии** — активные или завершённые сессии.
   - **Устройства** — модель, ОС, версия приложения.

## Что проверить

- [ ] Приложение заведено в AppMetrica, API Key скопирован.
- [ ] В коде выполнен вызов активации SDK с этим ключом.
- [ ] Приложение запущено, выполнены действия (навигация, экраны).
- [ ] В отчётах AppMetrica видны установки/сессии/устройства (с учётом задержки обновления).

## Что показать на экзамене / созвоне

1. Запустить приложение FinControl с подключённым AppMetrica, походить по экранам (Список, Обменник, Портфель).
2. Открыть кабинет AppMetrica → ваше приложение → **Отчёты** / **Онлайн**.
3. Показать, что появились сессии и/или устройства (с учётом задержки обновления данных).
4. Кратко сказать: «Добавил API key AppMetrica в приложение, запустил его и выполнил действия — в отчётах вижу сессии и устройство.»

## Устранение неполадок

- **Данные не появляются**: убедитесь, что API Key верный и что приложение реально вызывало активацию (без ошибок при старте). Проверьте фильтры по дате в отчётах.
- **Только тестовые данные**: для учебного проекта этого достаточно; в продакшене настройте обработку персональных данных по политике конфиденциальности.
