# Практика: Xcode — установка FinControl в симуляторе и логирование

## Цель

Открыть проект FinControl в Xcode, запустить приложение в симуляторе iOS и настроить просмотр логов (Console) для отладки и тестирования.

## Что понадобится

- macOS с установленным Xcode
- Проект FinControl с добавленной платформой iOS (папка `ios/`)

## Шаг 1: Открытие проекта в Xcode

1. В корне проекта FinControl откройте **не** `ios/Runner.xcodeproj`, а **`ios/Runner.xcworkspace`** (из-за CocoaPods/Flutter).
2. Двойной клик по `ios/Runner.xcworkspace` или из терминала:
   ```bash
   open ios/Runner.xcworkspace
   ```

## Шаг 2: Выбор симулятора и запуск

1. В Xcode вверху выберите схему **Runner** и в списке устройств — нужный симулятор (например iPhone 15, iOS 17).
2. Нажмите **Run** (▶) или `Cmd + R`.
3. Дождитесь сборки (Flutter подтянет зависимости при первой сборке). Приложение установится и запустится в симуляторе.

Альтернатива из терминала (из корня проекта):

```bash
flutter run
```

Выберите iOS-симулятор из списка.

## Шаг 3: Просмотр логов в Xcode

1. Во время работы приложения откройте **View → Debug Area → Activate Console** (или `Cmd + Shift + C`).
2. В нижней панели (Debug area) вкладка **Console** показывает вывод `print`, `debugPrint`, логи нативного кода и Flutter.
3. При выполнении действий в приложении (Обменник, Портфель, расходы) в консоли появляются сообщения.

## Шаг 4: Фильтрация логов

- В поле поиска консоли введите ключевое слово (например название экрана или тег, если он добавлен в коде).
- Для просмотра только ошибок ориентируйтесь на уровни вывода и стек-трейсы.

## Что проверить

- [ ] Открыт `ios/Runner.xcworkspace`, схема Runner, выбран симулятор.
- [ ] Приложение успешно собирается и запускается в симуляторе.
- [ ] В Console видны логи при взаимодействии с приложением.

## Дополнительно

- Установка на реальное устройство: выберите устройство в списке, настройте подписание (Signing & Capabilities) и при необходимости зарегистрируйте устройство в Apple Developer.
- Логи с устройства можно смотреть в **Window → Devices and Simulators** → выберите устройство → **Open Console**.
