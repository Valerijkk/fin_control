## Итог

- **Документация:** в шести файлах фич описан ожидаемый UI (кнопки, навигация, ошибки, загрузка, обратная связь); в PRD добавлены краткие UX-критерии; в критериях приёмки — отсылка к UX.
- **Код:** единый виджет `SectionTitle`, карточки переведены на `Card` из темы, поля ввода без дублирования стилей границы, на экране «Статистика» добавлена иконка настроек.
- **README/FAQ:** явно указано, где настраивается внешний вид приложения (`app_theme.dart`).

Дизайн с нуля не переписывался — только полировка формулировок и согласованность с описанным UX.
