# 📚 Документация FinControl — карта по папкам

> **FinControl** — учебное Flutter-приложение для практик мобильного тестирования: учёт расходов, обменник, портфель, интеграции.
> Всё настроено **из коробки**. Метрики — подставь токены в [STUDENT_ENV.md](STUDENT_ENV.md), Firebase — по инструкции в [10-firebase.md](practices/10-firebase.md).

| Папка | Назначение |

| 📋 **[acceptance-criteria/](acceptance-criteria/)** | Для экзаменатора: чек-листы по каждой практике, заложенные баги (задание со звёздочкой), что показать на созвоне |
| 📖 **[practices/](practices/)** | Практики по шагам: 00-getting-started, 01–09 (Charles, Proxyman, Android Studio, Xcode, ADB, Sentry, AppMetrica, TestFlight, Android-дистрибуция), 10-firebase (полный гайд). Порядок — в [practices-step-by-step.md](practices-step-by-step.md) |
| 🔧 **[technical/](technical/)** | Архитектура, модели данных, API-спецификация, запуск в Android Studio |
| 💼 **[business/](business/)** | PRD, майлстоуны, бизнес- и функциональные требования |
| 🧪 **[testing/](testing/)** | Тестовая документация: фичи (требования по экранам), тест-кейсы; ссылки на практики |
