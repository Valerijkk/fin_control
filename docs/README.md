## С чего начать (ученик)

| Документ | Для чего |
|----------|----------|
| **[practices-step-by-step.md](practices-step-by-step.md)** | Главный пошаговый гайд по практикам: клонирование, первый запуск, открытие в Android Studio, детальные шаги по практикам 01, 02, 03, 05, 06, 07, 00 (Firebase) и кратко — остальные. Читать и выполнять по шагам. |
