# Практика 13: Firebase Remote Config

## Технология

**Firebase Remote Config** — удалённые параметры (фичефлаги, строки, числа) без обновления приложения. Тестировщику нужно уметь добавлять параметры в консоли, получать их в приложении и проверять A/B или пошаговый rollout.

## В проекте FinControl

- После 00-firebase-setup подключается `firebase_remote_config`. Пример: параметр «включить экран Обменник» или «заголовок приветствия» — значение подставляется из консоли.

## Задание

1. Добавить в Firebase Console параметр (например булев или строковый).
2. Подключить Remote Config в приложении, получить значение при старте или при открытии экрана.
3. Использовать значение в UI (например показ/скрытие блока или текст).
4. Изменить значение в консоли, обновить в приложении (fetch) и убедиться, что UI обновился.
