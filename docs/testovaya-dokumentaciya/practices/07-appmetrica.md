# Практика 07: AppMetrica

## Технология

**AppMetrica** (Яндекс) — аналитика и мониторинг мобильных приложений: сессии, устройства, события. Тестировщику нужно уметь подключать SDK, отправлять события и проверять отчёты в кабинете.

## В проекте FinControl

- AppMetrica инициализируется опционально в `lib/config/telemetry.dart` (API Key). Приложение должно быть заведено в кабинете AppMetrica.

## Задание

1. Завести приложение в кабинете AppMetrica, получить API Key.
2. Подключить SDK в FinControl, передать API Key, инициализировать в `main.dart`.
3. Запустить приложение, выполнить несколько переходов по экранам.
4. Убедиться, что сессии/устройства отображаются в отчётах AppMetrica.

**Полная инструкция:** [docs/practices/07-appmetrica.md](../../practices/07-appmetrica.md).
