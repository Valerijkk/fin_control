# Практика 02: Proxyman

## Технология

**Proxyman** — альтернатива Charles для macOS/Windows: перехват HTTP/HTTPS, SSL-расшифровка, фильтры, breakpoints. Популярен у тестировщиков на Mac для сниффа трафика iOS и Android.

## В проекте FinControl

- Как и с Charles: трафик к API курсов (exchangerate.host, open.er-api.com) идёт по HTTPS при открытии Обменника и при работе с Портфелем.
- Настройка прокси на устройстве/симуляторе на компьютер с запущенным Proxyman; установка CA-сертификата для HTTPS.

## Задание

1. Установить Proxyman, настроить прокси на устройстве/симуляторе.
2. Установить сертификат Proxyman для расшифровки HTTPS.
3. Запустить FinControl, открыть экран с загрузкой курсов.
4. Проверить в Proxyman наличие запроса к API курсов и успешный ответ.
