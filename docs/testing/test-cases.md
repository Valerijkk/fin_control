# ✅ Тест-кейсы и автотесты FinControl

> Приложение **одно**, запускается **из коробки** (`flutter run`).
> Для проверки Sentry, AppMetrica и Firebase достаточно подставить ключи в указанный файл.

| Сервис | Где настраивать |
|--------|----------------|
| **Sentry и AppMetrica** | [`lib/config/student_env.dart`](../../lib/config/student_env.dart) — см. [STUDENT_ENV.md](../STUDENT_ENV.md) |
| **Firebase** | По инструкции в [10-firebase.md](../practices/10-firebase.md) |

В репозитории есть **шаблоны** тестов в `test/` (unit- и widget-тесты); остальной функционал покрыт ручными тест-кейсами ниже. При необходимости новые автотесты добавляются по тем же шаблонам.

---

## ✋ 2. Ручные тест-кейсы по фичам

---

### 2.1 📝 Учёт расходов (Home, Add/Edit)

| ID | Название | Предусловия | Шаги | Ожидаемый результат |
|----|----------|-------------|------|---------------------|
| M-1.1 | Добавление расхода | Shell → Список | Нажать «+», ввести сумму, название, категорию, сохранить | Запись появляется в списке |
| M-1.2 | Редактирование записи | В списке есть запись | Тап по записи → изменить → сохранить | Изменения отображаются |
| M-1.3 | Удаление с UNDO | В списке есть запись | Свайп влево → снэкбар → «ОТМЕНА» | Запись восстанавливается |
| M-1.4 | Фильтр по категории | Записи разных категорий | Выбрать чип «Еда» | Только записи категории Еда |
| M-1.5 | Быстрое добавление | Список открыт | Иконка быстрой записи → сумма → «Добавить» | Запись с дефолтным названием |

---

### 2.2 💱 Обменник

| ID | Название | Предусловия | Шаги | Ожидаемый результат |
|----|----------|-------------|------|---------------------|
| M-2.0 | Отображение курсов | Интернет включён | Открыть «Обменник» | Курсы отображаются (RUB/USD/EUR) |
| M-2.1 | Расчёт по курсу | Обменник открыт | Выбрать валюты, ввести сумму | Сумма пересчитывается по курсу |
| M-2.2 | Сохранение операции | Курсы загружены | Сумма > 0 → «Обменять» | Операция в истории обмена |
| M-2.3 | Валидация суммы 0 | Обменник открыт | Ввести 0 → «Сохранить» | Кнопка неактивна / ошибка (см. [баги](../acceptance-criteria/bugs-dlya-ekzamenatora.md)) |
| M-2.4 | Ошибка сети | Сеть выключена | Открыть Обменник | Сообщение об ошибке, без краша |
| M-2.5 | Оповещение по курсу | Курсы загружены | «Добавить оповещение» → настроить → «Добавить» | В списке; при срабатывании — SnackBar |
| M-2.6 | Отложенный обмен | Курсы загружены | «Добавить отложенный обмен» → настроить | Ордер в «Ожидание»; при исполнении — запись в истории |

---

### 2.3 💼 Портфель

| ID | Название | Предусловия | Шаги | Ожидаемый результат |
|----|----------|-------------|------|---------------------|
| M-3.1 | Отображение баланса | Портфель открыт | — | Баланс в базовой валюте |
| M-3.2 | Покупка актива | Баланс > 0 | Купить → валюта, сумма → подтвердить | Баланс ↓, позиция появляется |
| M-3.3 | Продажа актива | Есть позиция | Продать → сумма → подтвердить | Позиция ↓, баланс ↑ |
| M-3.4 | История сделок | Есть сделка | Прокрутить до истории | Тип, валюта, сумма, дата |
| M-3.5 | Доходность (PnL) и ROI | Есть позиции, курсы загружены | Прокрутить до «Доходность» | Стоимость, cost basis, PnL, ROI % |
| M-3.6 | Аллокация активов | Есть позиция | Открыть Портфель | Доли в % и полоски по валютам |
