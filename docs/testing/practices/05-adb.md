# Практика 05: ADB

## Технология

**ADB (Android Debug Bridge)** — консольная утилита для управления устройством/эмулятором Android: установка/удаление приложений, логи, скриншоты, очистка данных, shell. Необходима тестировщику для автоматизации и ручных проверок без Android Studio.

## В проекте FinControl

- Установка APK: `adb install -r build/app/outputs/flutter-apk/app-debug.apk`.
- Очистка данных приложения: `adb shell pm clear <package>` — сброс к состоянию «как после установки» (в т.ч. SQLite, SharedPreferences).
- Логи: `adb logcat` с фильтром по пакету/тегу.
- Скриншоты: `adb exec-out screencap -p > screen.png`.

## Задание

1. Установить FinControl через ADB на эмулятор или устройство.
2. Получить логи приложения (logcat с фильтром).
3. Сделать скриншот экрана устройства.
4. Очистить данные приложения и убедиться, что при следующем запуске приложение в начальном состоянии.
