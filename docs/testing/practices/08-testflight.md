# Практика 08: TestFlight

## Технология

**TestFlight** — сервис Apple для распространения бета-сборок iOS среди тестировщиков. Нужно уметь собирать архив, загружать в App Store Connect и добавлять тестеров.

## В проекте FinControl

- Сборка iOS-приложения (Flutter): подписание, архив, загрузка в App Store Connect, распространение через TestFlight для внутренних/внешних тестеров.

## Задание

1. Настроить подписание (Apple Developer, сертификаты, provisioning profile).
2. Собрать архив (Xcode или `flutter build ipa`).
3. Загрузить сборку в App Store Connect и опубликовать в TestFlight.
4. Добавить тестера и установить приложение через TestFlight.

**Полная инструкция:** [docs/practices/08-testflight.md](../../practices/08-testflight.md).
