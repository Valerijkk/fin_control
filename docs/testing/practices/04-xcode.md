# Практика 04: Xcode

## Технология

**Xcode** — среда разработки для iOS: симулятор, сборка и установка приложений, консоль логов (Console). Тестировщику нужно уметь запускать приложение в симуляторе и смотреть логи.

## В проекте FinControl

- Проект имеет платформу `ios/` (Flutter). Запуск через `flutter run` с выбором симулятора или открытие `ios/Runner.xcworkspace` в Xcode.
- Логи выводятся в консоль Xcode при запуске приложения из Xcode или при подключении к процессу.

## Задание

1. Открыть симулятор iOS, собрать и запустить FinControl (`flutter run` или сборка из Xcode).
2. Выполнить действия в приложении (обменник, портфель, список) и убедиться, что логи видны в Console.

**Полная инструкция:** [docs/practices/04-xcode.md](../../practices/04-xcode.md).
