# Практика 11: Firebase FCM

## Технология

**Firebase Cloud Messaging (FCM)** — push-уведомления. Тестировщику нужно уметь получать токен устройства, отправлять тестовые сообщения из консоли и проверять отображение уведомлений в приложении.

## В проекте FinControl

- После 00-firebase-setup добавляется `firebase_messaging`: получение токена, обработка сообщений в foreground/background. Токен можно вывести в настройках или логи для тестовой отправки.

## Задание

1. Подключить Firebase Messaging в проекте, запросить разрешения (iOS/Android).
2. Получить FCM-токен и вывести его (логи или экран настроек).
3. Отправить тестовое сообщение из Firebase Console (Cloud Messaging) на этот токен.
4. Убедиться, что уведомление приходит и отображается (в foreground/background по реализации).

**Полная инструкция:** [docs/practices/11-firebase-fcm.md](../../practices/11-firebase-fcm.md).
