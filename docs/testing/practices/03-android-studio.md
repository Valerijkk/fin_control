# Практика 03: Android Studio

## Технология

**Android Studio** — среда разработки для Android: эмулятор (AVD), сборка и установка приложений, просмотр логов (logcat), отладка. Тестировщику нужно уметь запускать приложение в эмуляторе и фильтровать логи по тегу/приложению.

## В проекте FinControl

- Flutter-проект собирается в APK и запускается на эмуляторе через `flutter run` или установку готового APK.
- Логи приложения видны в logcat (тег/имя процесса можно фильтровать по названию приложения).

## Задание

1. Создать и запустить AVD в Android Studio.
2. Установить FinControl из проекта (`flutter run`) или установить собранный APK.
3. Настроить logcat: фильтр по приложению/тегу, воспроизвести действие в приложении и убедиться, что логи видны.
