# Критерии приёмки: 01 — Charles Proxy

## Обязательно

- [ ] Charles установлен и запущен, прокси на устройстве/эмуляторе настроен (хост + порт 8888).
- [ ] Сертификат Charles установлен на устройстве и помечен как доверенный (для HTTPS).
- [ ] В Charles включён SSL Proxying для хостов API курсов (или *:443).
- [ ] Приложение FinControl открыто, выполнен переход на экран с загрузкой курсов (Обменник или главный с карточкой курсов).
- [ ] В Charles виден хотя бы один запрос к `api.exchangerate.host` или `open.er-api.com`.
- [ ] Ответ на запрос — HTTP 200, в теле JSON с полями курсов (rates или аналог).

## Желательно

- [ ] Ученик продемонстрировал Breakpoint или Throttling (по заданию или дополнительно).
