# Не открывай эту папку в Android Studio как проект

Это только нативная часть Flutter-приложения.

**Открой в Android Studio папку на уровень выше** — корень проекта `fin_control` (там, где лежит `pubspec.yaml`). Тогда проект откроется как Flutter-приложение и всё заработает.
